# Sand Sort Lab Site

Static GitHub Pages site for Sand Sort Lab.

## Pages

- `index.html`: entry page that redirects to the privacy policy.
- `privacy.html`: public privacy policy URL for App Store Connect.
- `support/index.html`: public technical support URL for App Store Connect.

## GitHub Pages Setup

In the repository settings, enable GitHub Pages with:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

After GitHub finishes publishing, the privacy policy should be available at:

`https://velvetrogue5.github.io/sand-sort-lab-site/privacy.html`

The technical support page should be available at:

`https://velvetrogue5.github.io/sand-sort-lab-site/support/`

Update `privacy.html` before release if the app adds advertising, analytics, in-app purchases, cloud saves, customer support collection, or any other data collection.
